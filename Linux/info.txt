GNOME  KDE  XFCE

========= GNOME =========


GNOME (GNU Network Object Model Environment) - свободная среда рабочего стола для UNIX-подобных операционных систем. GNOME не является частью проекта GNU. Размер: 500-700 МБ
Центральную роль в GNOME играет инструментарий GTK+

========= KDE =========


KDE (K Desktop Environment) - набор программных компонентов, предназначенных для создания пользовательского интерфейса. Основная среда рабочего стола - Plasma Desktop. Размер: 600-900 МБ
KDE Plasma является основной средой рабочего стола на многих дистрибутивах Linux

========= XFCE =========


XFCE (XForms Common Environment) - это легковесная среда рабочего стола для Linux, которая известна своим низким потреблением ресурсов. Размер: 300-400 МБ
Оливье Фордан начал проект Xfce в 1996 году
